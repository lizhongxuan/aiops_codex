目标口径：

- 参考 Claude Code，把 `aiops-codex` 的 prompt 组装收敛成统一骨架
- 将 tool 使用说明收回到 tool-owned prompt
- 将“证据门槛 / 来源数量 / 回答样式 / 是否允许早停”下沉到结构化 `TurnPolicy`
- 删除重复的 market snapshot prose
- 删除未参与实际 prompt assembly 的 dead prompt builder
- 保留必要的 legacy 恢复路径，但把删除窗口明确写清
