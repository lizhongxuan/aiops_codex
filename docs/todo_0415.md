## 1. 总体验收口径

- [ ] 新会话默认进入 `分析模式`，不得隐式进入执行态。
- [ ] 任意关键结论都能关联至少一个 `EvidenceRecord`。
- [ ] 未通过计划审批前，mutation 工具和危险命令不得落地执行。
- [ ] 高风险动作执行前必须展示动作审批卡，并能看到目标、风险、影响面和回滚提示。
- [ ] 用户点击停止后，取消信号能向下传播到 loop、orchestrator、host-agent 和外部请求。
- [ ] 执行动作后必须自动进入验证链路，不得只返回“命令已执行”。
- [ ] 协作工作台会话结束后能查看 incident timeline、审批记录、证据摘要和验证结果。
